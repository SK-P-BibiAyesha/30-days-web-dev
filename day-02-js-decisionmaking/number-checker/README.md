# Number Checker

A simple JavaScript project that checks whether a number is Positive, Negative, or Zero.

## Features

- User enters a number
- Checks if the number is:
  - Positive 😊
  - Negative 😔
  - Zero 😐
- Displays result dynamically
- Input validation for empty fields
- Responsive card-based UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

1. Enter a number in the input field.
2. Click the **Check Number** button.
3. The application determines whether the number is:
   - Positive
   - Negative
   - Zero
4. The result is displayed instantly on the page.
